    testUnit() calls alternating union() and find() commands between random numbers. It takes a UnionFind object that has been initialized to the specified size:

    def testUnit(object)
  @size.times do
    object.union(rand(@size), rand(@size))
    object.find(rand(@size), rand(@size))
  end
end
